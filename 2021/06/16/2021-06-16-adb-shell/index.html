adb 常用命令

&emsp;

  1. 查看所有包：abd shell pm list packages
  2. 查看禁用的包：adb shell pm list packages -d
  3. 查看启用的包：adb shell pm list packages -e
  4. 禁用某个包：adb shell pm disable-user 包名
  5. 启用某个包：adb shell pm enable 包名
