二、使用 VS Code 生成格式化代码：

在 VS Code 中打开你的代码文件，然后安装对应编程语言的插件，插件生效后，复制你的代码，粘贴到 Word 中，粘贴选项选择保留源格式即可。

因为 VS Code 和 Word 都是微软公司的，所以 VS Code 中的代码复制到 Word 后，会自动进行排版，无需自行调整。
