全局更新：

更新 Git：https://git-scm.com/download/win

更新 Node.js：https://nodejs.org/en/download

更新 npm：npm install -g npm

更新 npm-check：npm install -g npm-check

使用 npm-check更新所有包：npm-check -u -g

博客更新：

在博客目录下执行：npm update --save hexo-theme-fluid

下载：https://github.com/fluid-dev/hexo-theme-fluid/releases

解压后改名为 fluid，再放到 themes 中

常用命令：

hexo clean
（hexo server）
hexo g -d
